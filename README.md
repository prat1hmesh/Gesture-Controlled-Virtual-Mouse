A beginner-friendly computer vision project that allows users to control mouse movement and clicks using hand gestures through a webcam.

## Features
- Real-time hand tracking
- Cursor movement using index finger
- Left click using thumb + index gesture
- Visual hand landmark feedback

## Technologies Used
- Python
- OpenCV
- MediaPipe
- PyAutoGUI
- NumPy

## How to Run
1. Install dependencies:
   pip install opencv-python mediapipe pyautogui numpy

2. Run the program:
   python virtual_mouse.py
